2 # ROS Beginner Tutorials
3 
4 A beginner ROS package that implements simple publisher-subscriber nodes in ROS
5 
6 
7 ## Installations
8 
9 To install ROS refer [this link](http://wiki.ros.org/ROS/Installation).
10 
11 Preferred Ubuntu 18.04, ROS Melodic
12 
13 
14 ### Step 1. Clone repository and build the package
15 ```bash
16 git clone --recursive https://github.com/kavyadevd/beginner_tutorials.git
17 cp <repository_path> <catkin_workspace_path/src/>
18 cd <catkin_workspace>
19 source ./devel/setup.bash
20 catkin_make
21 ```
22 
23 ### Step 2. Make sure that a roscore is up and running
24 ```bash
25 roscore
26 ```
27 ### Step 3. In new terminal run publisher node
28 ```bash
29 rosrun beginner_tutorials talker
30 ```
31 ### Step 4. In new terminal run subscriber node
32 ```bash
33 rosrun beginner_tutorials listener
34 ```
35 
36 #### To run launch file
37 
38 ```bash
39 roslaunch beginner_tutorials LaunchTalkerListener.launch frequency:=<desired-frequency>
40 ```
41 This command will launch published and subscriber.
42 
43 #### To run service
44 ```bash
45 rosservice call /ServiceFile "<desired-custom-message>"
46 ```
47 
48 #### To invoke rqt logger and console on separate terminals run the following two commands
49 ```bash
50 rosrun rqt_console rqt_console
51 rosrun rqt_logger_level rqt_logger_level
52 ```

55 ## Plugins

57 
58 - CppCheckEclipse
59 
60  To install and run cppcheck in Eclipse
61 
62  1. In Eclipse, go to Window -> Preferences -> C/C++ -> cppcheclipse.
63  Set cppcheck binary path to "/usr/bin/cppcheck".
64 
65  2. To run CPPCheck on a project, right-click on the project name in the Project Explorer
66  and choose cppcheck -> Run cppcheck.
67 
68  3. To run on terminal
69  ```bash
70  cppcheck --enable=all --std=c++11 -I include/ --suppress=missingIncludeSystem $( find . -name *.cpp -or -name *.h | grep -vE -e "^./build/" -e "^./vendor/") > Results/cppcheckoutput.xml
71  ```
72  Results are present at Results/cppcheckoutput.xml
73 
74 - Cpplint
75  1. To run cpplint on terminal
76  ```bash
77  cpplint $( find . -name *.cpp | grep -vE -e "^./build/" -e "^./vendor/") $( find . -name *.hpp | grep -vE -e "^./build/" -e "^./vendor/") > Results/cpplintoutput.txt
78  ```
79  Results are present at Results/cpplintoutput.xml

83  To include and use Google C++ Style formatter in Eclipse
84 
85  1. In Eclipse, go to Window -> Preferences -> C/C++ -> Code Style -> Formatter.
86  Import [eclipse-cpp-google-style][reference-id-for-eclipse-cpp-google-style] and apply.
